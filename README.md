# UniPay – Smart Campus Cashless Payment Ecosystem

UniPay is a smart campus cashless payment ecosystem designed to simplify and secure financial transactions within educational institutions. The platform enables students to make payments using NFC-enabled ID cards, eliminating the need for physical cash while providing a fast, convenient, and transparent payment experience. The system connects students, merchants, and administrators through a unified platform that supports real-time transaction processing, wallet management, and campus-wide payment services.

Frontend Design & Features

Built with Flutter, the UniPay frontend delivers a modern, responsive, and user-friendly experience through dedicated role-based interfaces for Students, Merchants, and Administrators.

The Student Dashboard provides real-time wallet balance, NFC card status, recent transactions, and quick actions such as adding funds, viewing payment history, and managing card settings. The interface is designed to give users complete visibility and control over their campus spending.

The Merchant Dashboard allows vendors to monitor earnings, track incoming NFC payments, and access transaction records through a clean and intuitive interface, ensuring efficient payment management and transparency.

The Admin Console offers centralized system management, enabling administrators to manage users, onboard merchants, oversee NFC card operations, and monitor platform activities through organized management modules.

UniPay also includes a detailed Transaction Management System that displays payment information, transaction status, timestamps, and unique transaction identifiers for enhanced transparency and record keeping. Additionally, the Profile and Security Module allows users to manage account details, configure security settings, and protect access through PIN-based authentication.
